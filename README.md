# Intercom Moto (Android nativo)

App Kotlin nativo para comunicação entre piloto e passageiro de moto via WebSocket,
com ativação de voz por palavra-gatilho "AVISO".

## Como subir pelo Termux

```bash
cd IntercomMoto
git init
git add .
git commit -m "Intercom Moto - app nativo Kotlin"
git branch -M main
git remote add origin https://github.com/ftomazelajr/comunicador.git
git push -u origin main
